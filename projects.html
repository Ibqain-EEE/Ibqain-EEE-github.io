Projects

Here are a few of the projects I have worked on. If you would like to know more about them or would like to see my resume, please free to contact me on Issabqain@hotmail.com or Ib818@ic.ac.uk .

Visual Access For Blind People (B.E.L.T.S)

Unfortunately, the blind are at a disadvantage when it comes to academic attainment simply since the necessary tools needed are not widely available. The focus of this project was to provide an Imaginative, innovative and cost-effective system to combat this.

Electronic Charity Box

During a Christmas charity fundraising run with Greenford Rotary Club, I noticed that an excessive amount of time was taken up to count coin donations. Therefore, to allow fundraisers to spend more time fundraising rather than counting coins I built from scratch a protoype Charity box to reduce time taken to count coin donations

Intruder Alarm

Developed an alarm designed for domestic use to alert users when a certain area has been tresspased

Robotic Arm Feeder

SpikeGuard

An embedded IoT device to help reduce drink spiking

2048 Game

Made the viral 2048 game using C++

Mastermind Algorithm

Wrote an algorithm to solve any mastermind code with as many symbols or digits.

Desinging an Operational Amplifier

Designed an operational amplifier using LT072 and MC33078 Chips to meet rigorous criteria

Signal Detecting Rover

Designed a wirelessly controlled rover with the ability to sense, store and decode data for Acoustic, Infrared and Radio signals carrying UART data. As the project manager, I also distributed tasks amongst the team, set deadlines, maintained documentation and edited the technical report
